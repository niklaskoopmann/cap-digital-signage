# Xibo Media Sync

This repository packages a terminal-based sync tool for copying local media into a Xibo CMS library.

Start here:

- [User Guide](docs/USER_GUIDE.md) for installation, running the sync, and Docker setup.
- [Technical Documentation](docs/TECHNICAL.md) for repository structure, sync flow, configuration, and extension notes.

Scripts:

- `scripts/sync_xibo_no_wifi.py`: sync only, no Wi‑Fi handling.
- `scripts/sync_xibo.py`: sync plus Windows Wi‑Fi hotspot connection.
